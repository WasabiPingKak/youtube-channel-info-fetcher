# 工具函數說明文件

本目錄包含各種通用工具函數和輔助功能。以下是各個工具的詳細說明：

## 影片分類工具

### categorizer.py
- **用途**：影片分類邏輯的實現
- **主要功能**：
  - 影片內容分析
  - 分類規則應用
  - 分類標籤生成

## YouTube 相關工具

### youtube_utils.py
- **用途**：YouTube API 相關的工具函數
- **主要功能**：
  - YouTube API 參數處理
  - 影片資訊解析
  - 頻道資訊處理
  - API 回應格式化

## 使用說明

這些工具函數主要供服務層（services）使用，提供以下功能：
1. 資料處理和轉換
2. API 調用輔助
3. 通用功能封裝
4. 錯誤處理輔助

## 注意事項

- 工具函數應該保持獨立性，避免與業務邏輯耦合
- 確保函數的可重用性和可測試性
- 適當處理異常情況
- 提供清晰的文檔註釋
